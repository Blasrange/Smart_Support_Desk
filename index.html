Sobre Mí

🎫 Sistema de Tickets

Crea, gestiona y responde tickets en tiempo real. Soporte por agentes o automáticamente con IA.

🤖 Asistente Inteligente (IA)

Integración con la API de IA para respuestas automatizadas y escalamiento a agentes humanos si es necesario.

🔔 Notificaciones y Tiempo Real

Notificaciones por correo electrónico y WebSockets en vivo usando Laravel Echo y Pusher.

📂 Archivos y Exportación

Adjunta archivos a los tickets y exporta conversaciones completas en PDF para respaldo o auditoría.

⚙️ Backend Laravel 12

Arquitectura basada en modelos sólidos, colas para procesamiento en segundo plano y API RESTful.

🧩 Frontend Vue 3 + Inertia.js

Aplicación SPA moderna y responsiva con TailwindCSS, componentes reutilizables y composición limpia.

🚀 Despliegue Profesional

Preparado para producción con Nginx, Supervisor, Certbot SSL y configuración `.env` optimizada.
